Utility

These functions provide shortcuts for common operations.

draw_self_floored()
Draws the instance at a floored coordinate instead of fractionary.
ds_list_equal(list1,list2)
Compares two ds lists to see if their contents match. Returns true or false.
ds_map_get(map,key)
Gets key from map. If the key doesn't exist, undefined is returned.
ds_map_set(map,key,value)
Sets key from map to value. If the key already exists, it is replaced.
instance_destroy_id(instance)
Destroys the instance.
instance_some(object)
Returns a random instance of the object.
mouse_in_window()
Returns whether the mouse cursor is currently inside the game window.
outside_room()
Returns whether the instance is completely outside the room region.
event_draw()
Calls the instance's Draw event.
event_step()
Calls the instance's Step event.
pick(which,opt1,opt2,...)
Returns which one option, up to 15 options.
instance_create_moving(x,y,object,speed,direction)
Creates an instance of object at x,y moving with speed and direction. The speed and direction values are given to the instance before the Create event is executed.
