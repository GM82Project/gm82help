Math

Shortcuts for common math operations.

ceilto(val,to)
Rounds up val to the nearest multiple of to.
floorto(val,to)
Rounds down val to the nearest multiple of to.
roundto(val,to)
Rounds val to the nearest multiple of to.
cosine(a,b,amount)
Cosine ease from a to b by specified amount.
esign(val,default)
Returns the default argument when the sign of val is zero. Example: 
    
image_xscale = esign(hspeed, image_xscale);
Here, image_xscale will be kept if the hspeed is zero, preventing the instance from becoming flat.
gauss(range)
Returns a gaussian distribution random value within the supplied range.
in_range(val,min,max)
Returns whether val is inbetween min and max.
inch(val,goto,stepsize)
Approximates val to goto by stepsize until they match exactly.
irandom_fresh(oldval,min,max)
randomizes an integer within min and max range without repeating oldval.
modwrap(val,min,max)
Wraps val around both directions inside a box defined by min and max. Lower bound is inclusive, upper bound is exclusive, like a modulo.
real_hex(string)
Converts a hex string into a real.
