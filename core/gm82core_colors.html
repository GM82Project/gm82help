Colors

Functions for dealing with color values.

color_blend(col1,col2)
Multiplies col1 with col2 and returns the color.
color_get_luminance(color)
Returns a Kodak-corrected luminance value from color (0-255).
merge_color_corrected(col1,col2,factor)
Merges col1 towards col2 by factor. Uses corrected square formula for more natural results.
color_reverse(color)
Reverses the blue and red components of color (works both ways).
