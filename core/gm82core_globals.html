Globalvars

These values are updated for you, and can be used in code.

delta_time*
Number of milliseconds since last frame.
fps_fast*
Fast FPS estimate based on well-known FRAPS algorithm.
fps_real*
Same as Studio - measures theoretical maximum fps based on one second divided by the time it took to complete the last game step.
windows_version*
Windows version as a number - 5 = XP, 6 = Vista, 7 = Seven, 8 and later return 8.
