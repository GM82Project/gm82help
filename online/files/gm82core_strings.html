Strings

Functions for handling strings and encoding.

base64_decode(str)
Decodes a base64 string into raw bytes. Returns a string.
base64_encode(str)
Encodes a string into base64.
date_get_timestamp([date])
Returns a standard timestamp string from a date (by default, the current date is used). Format is: 
dd/mm/yy hh:mm
string_better(val)
Regular string() but displaying up to 8 decimal digits.
string_hex(real)
Converts a real into a hex string.
string_number(string)
A version of string_digits() that also keeps the minus sign and fractionary dot.
string_ord_at(str,pos)
ord(string_char_at(str,pos))
string_pad(number,digits)
Pads the number to have the number of digits.
string_token_start(str,sep)
Starts splitting str by sep. You can only split one string at a time.
string_token_next()
Returns the next bit of text before the next sep set up by the previous function, or an empty string when done.
strong(val1,val2,val3...)
Concatenates the arguments as strings and returns it.
