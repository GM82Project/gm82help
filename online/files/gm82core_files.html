Files

Functions for dealing with files.

ds_map_read_ini(map,filename)
Reads ini file at filename into map. Section names are prepended to keys, separated by a space.
directory_previous(dir)
Returns the parent directory of dir.
file_size(filename)
Returns the size of file filename in bytes. If the file doesn't exist, -1 is returned.
file_text_read_all(fname,[separator])
Reads file filename in its entirety and returns it as a string. If a separator string is provided, it is used inbetween lines, otherwise CR LF is used.
filename_remove_ext(fn)
Removes everything after the dot from a string. Note: this does not rename the file.
