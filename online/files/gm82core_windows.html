Windows

Functions for interacting with the operating system.

get_delta()
Gets the time since the last call of this function in microseconds.
mouse_back_button()
Returns if the back button is pressed on mice with navigation controls.
mouse_forward_button()
Returns if the forward button is pressed on mice with navigation controls.
mouse_check_direct(button)
Directly checks the hardware for updated mouse button states.
mouse_in_window()
Returns whether the mouse cursor is currently inside the game window.
registry_read_dword(key,[default])
Reads a REG_DWORD from the specified key. Returns default if the value is not found.
registry_write_dword(key,value)
Writes a REG_DWORD to the specified key.
set_working_directory(dir)
Changes working_directory to the specified directory.
sleep_ext(ms)
Sleeps ms milliseconds without updating keyboard and mouse state.
window_get_caption_color()
Returns the Windows 8+ user accent color. If the accent color is disabled in Windows theme preferences, white is returned. Returns black in Windows 7 and older.
window_has_focus()
Returns whether the game window is currently in front and active.
window_minimize()
Minimizes the game window. Will reset the application title to room_caption.
